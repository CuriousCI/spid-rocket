## Run example 

The binary example is written in ```src/```, the *spid* library in ```lib/```, check the ```Cargo.toml``` for more info.
To run the the project:

```
git clone https://github.com/CuriousCI/spid-rocket.git
cd spid-rocket
cargo run
```
